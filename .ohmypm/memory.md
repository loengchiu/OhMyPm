# 项目记忆

## 1. 项目概览
- 项目名称：Oh My PM
- 当前需求任务：以“审批路径扩展”为样例，验证 OhMyPm 最小主链路是否能从自然语言输入走到原型、PRD、评审和状态回写。
- 当前模式：alignment_loop
- 当前版本：v0.6

## 1.1 协作边界
- 当前审批路径扩展链路是机制验证样例，不是真实项目协作线程。
- 当前样例用于验证自然语言路由、门禁、preflight、交付分工、评审和状态回写是否能连起来。
- PM 不需要回答样例中的虚拟业务问题，例如金额阈值、风险条件、复核角色映射。
- 如果后续转入真实项目协作，必须单独开启真实上下文，不能把样例假设直接当成真实事实继承下去。
